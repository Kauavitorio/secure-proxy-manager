Project Information

Secure Proxy

A transparent proxy solution with security features and a modern web UI

Features
  • Transparent HTTP/HTTPS proxy based on Squid
  • IP address blacklisting (local and remote)
  • Domain name blacklisting (local and remote)
  • Direct IP access blocking with exceptions
  • Detailed proxy access logs
  • Modern web interface for configuration and monitoring
  • Dockerized deployment for easy setup
Technology Stack
  • Proxy Engine: Squid
  • Backend API: Python Flask
  • Frontend: HTML5, CSS3, JavaScript
  • UI Framework: Bootstrap 5
  • Database: SQLite
  • Containerization: Docker & Docker Compose
Getting Started

To use this proxy, configure your network devices to use this server as their proxy or set up your router to redirect all HTTP/HTTPS traffic through this proxy.
